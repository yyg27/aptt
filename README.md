
# aptt

Interactive package installer for Debian/Ubuntu based Linux systems with fuzzy search. 
## Installation

### Quick Install
```bash
curl -sL https://raw.githubusercontent.com/yyg27/aptt/main/install.sh | bash
```

## Requirements

- `apt (Standart on Debian/Ubuntu)`
- `fzf`

## Usage
For INSTALLING packages
```bash
aptt
```
For REMOVING already installed packages
```bash
apttrm
```

- Type to search
- **Tab** to select multiple packages
- **Enter** to install/remove
- **ESC** to cancel
